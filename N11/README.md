В этом проекте мы будем создавать расширяющиеся и сворачивающиеся секции.
Эта функциональность может быть найдена в аккордеонах или дисплеях с переключением.


## Использованные технологии

* typescript
* react на хуках
* effector
* webpack
* eslint
